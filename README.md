# Todolist

A Cupertino-style task list with calendar scheduling.

## Features
- Add errands with date and time
- Weekly calendar with event dots
- Filter errands by selected day
- Dark mode toggle
- List font color picker
- Archive and restore tasks

## Run
```
flutter pub get
flutter run
```
